# Machine Learning Classification Model

This project implements a machine learning pipeline using the **Random Forest Classifier** to classify data based on a given dataset. It includes data preprocessing, model training, and evaluation metrics to assess performance.

## 🧠 Project Structure

1. **Load the Dataset**  
   Uses pandas to load and analyze the dataset.

2. **Data Preprocessing**  
   - Conversion of data types  
   - Encoding categorical variables  
   - Handling missing values

3. **Model Building**  
   - Random Forest Classifier from `sklearn`  
   - Train-Test Split for validation

4. **Model Evaluation**  
   - Accuracy Score  
   - Classification Report  

## 🛠️ Tech Stack

- Python 3.x
- pandas
- scikit-learn
